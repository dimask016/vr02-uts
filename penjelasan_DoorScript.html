Analisis Detail Skrip Mekanis Pintu Angkat Vertikal (DoorScript.cs)

Halaman ini berisi penjelasan lengkap baris demi baris dari skrip DoorScript.cs. Dokumentasi ini dibuat untuk mempermudah pemahaman logika dasar perpindahan translasi posisi pintu satu arah (vertikal ke atas) secara absolut dan dinamis di Unity.

1. Library/Namespace Imports

using UnityEngine;

Mengimpor pustaka (library) utama engine:

  • using UnityEngine; : Mengakses fungsionalitas dasar Unity seperti komponen Transform, struktur data matematika Vector3, pencatatan log debug, serta perhitungan frame waktu.

2. Deklarasi Properti Konfigurasi Pintu

public class DoorScript : MonoBehaviour
{
    [Header("Pengaturan Pintu")]
    public Transform doorMesh;
    public float openHeight = 5f; // Tinggi pintu saat terbuka (meter)
    public float openSpeed = 3f;

Mendefinisikan parameter mekanis pintu yang dapat diubah nilainya melalui Inspector Unity:

  • doorMesh : Slot referensi objek 3D model pintu yang akan digeser naik-turun.
  • openHeight : Menentukan jarak pergeseran vertikal maksimal pintu saat membuka (dalam contoh ini diatur 5 meter ke atas).
  • openSpeed : Parameter pengatur kecepatan gerak geser pintu.

3. Variabel Posisi Internal & Status

    private Vector3 closedPosition;
    private Vector3 openPosition;
    private bool isOpen = false;

Variabel privat yang digunakan secara internal untuk memproses posisi koordinat:

  • closedPosition : Menyimpan titik koordinat 3D awal pintu ketika dalam keadaan tertutup rapat.
  • openPosition : Menyimpan titik koordinat tujuan akhir ketika pintu terangkat ke atas.
  • isOpen : Variabel penanda status (*state boolean*) untuk melacak apakah pintu sedang aktif membuka (true) atau menutup (false).

4. Penguncian Koordinat Absolut (Fungsi Start)

    void Start()
    {
        if (doorMesh != null)
        {
            // Kunci posisi awal dan posisi akhir sejak awal game secara mutlak
            closedPosition = doorMesh.position;
            openPosition = closedPosition + new Vector3(0, openHeight, 0);
        }
        else
        {
            Debug.LogError("PENTING: Tarik objek Pintu ke kolom Door Mesh!");
        }
    }

Melakukan inisialisasi koordinat target saat game pertama kali dijalankan:

  • if (doorMesh != null) : Memastikan validasi bahwa objek pintu sudah dipasang di Inspector agar tidak terjadi error *NullReferenceException*.
  • closedPosition = doorMesh.position; : Mengunci koordinat posisi letak awal pintu di dalam map sebagai acuan menutup.
  • openPosition = closedPosition + new Vector3(0, openHeight, 0); : Menghitung posisi target buka dengan cara menambahkan nilai sumbu Y posisi awal dengan jarak openHeight. Pintu akan bergeser lurus ke atas.
  • Debug.LogError(...) : Menampilkan pesan error berwarna merah di konsol Unity jika developer lupa memasukkan objek pintu.

5. Interpolasi Pergerakan Real-Time (Fungsi Update)

    void Update()
    {
        if (doorMesh == null) return;

        // Tentukan ke mana arah tujuan posisi secara real-time
        Vector3 target = isOpen ? openPosition : closedPosition;

        // Paksa objek bergerak ke target secara absolut
        doorMesh.position = Vector3.MoveTowards(doorMesh.position, target, openSpeed * Time.deltaTime);
    }

Menggerakkan objek pintu secara halus di setiap frame berjalan:

  • Vector3 target = isOpen ? openPosition : closedPosition; : Menggunakan operasi *Ternary* untuk menentukan titik koordinat tujuan. Jika isOpen bernilai true, targetnya adalah posisi di atas (openPosition), jika false targetnya adalah posisi semula (closedPosition).
  • Vector3.MoveTowards(...) : Memindahkan posisi pintu dari koordinat saat ini menuju koordinat target secara linear dengan kecepatan konstan.
  • openSpeed * Time.deltaTime : Menjaga agar pergerakan transisi pintu tetap mulus, stabil, dan kecepatannya konsisten di berbagai perangkat tanpa terpengaruh oleh naik-turunnya performa FPS (Frame Per Second) game.

6. Sakelar Pembalik Status (Fungsi ToggleDoor)

    // Fungsi yang dipanggil oleh Button
    public void ToggleDoor()
    {
        isOpen = !isOpen;
        Debug.Log("Tombol diklik! Status pintu saat ini: " + (isOpen ? "TERBUKA" : "TERTUTUP"));
    }
}

Fungsi pemicu interaksi utama:

  • public void ToggleDoor() : Fungsi ini dideklarasikan secara publik agar dapat dihubungkan dan dipanggil oleh komponen tombol UI (Button onClick) atau dipicu lewat tembakan laser raycast karakter.
  • isOpen = !isOpen; : Membalikkan kondisi boolean saat ini (*toggle*). Jika pintu sedang menutup, maka perintah ini akan mengubah statusnya menjadi membuka, dan begitu pula sebaliknya.
  • Debug.Log(...) : Mengirimkan pesan konfirmasi status operasional pintu terbaru ke jendela Console Unity untuk mempermudah proses pengujian (*debugging*) oleh developer.
