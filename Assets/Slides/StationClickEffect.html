🎨 StationClickEffect.cs

🎨 Visual Effect · Runtime · 37 baris

Fungsi utama: Mengontrol efek visual pada stasiun (objek 3D) dengan mengubah warna material dan menambahkan efek emisi (glow). Digunakan oleh EvacuationManager untuk menandai stasiun yang sudah selesai (berubah menjadi hijau) atau mereset ke warna aslinya. Memberikan umpan balik visual yang jelas kepada pemain tentang progres evakuasi.

📌 Baris Kode (cuplikan) 🎯 Fungsi / Metode 📖 Penjelasan
public class StationClickEffect : MonoBehaviour Deklarasi Kelas Kelas turunan MonoBehaviour yang dapat dipasang pada objek stasiun (kubus berwarna) untuk mengelola efek visualnya. Memberikan API publik untuk mengubah warna dan emisi.
public Color originalColor = Color.white;
public string stationName = "";		 Variabel Publik originalColor – warna awal stasiun (dapat diatur di Inspector). Digunakan sebagai referensi untuk ResetColor().
stationName – (tidak digunakan dalam kode) dapat digunakan untuk identifikasi atau debugging.
private Renderer rend;
private Material mat;		 Variabel Privat rend – referensi ke komponen Renderer objek.
mat – referensi ke Material yang digunakan oleh renderer. Disimpan agar dapat diubah langsung tanpa perlu GetComponent setiap kali.
void Start()
{
rend = GetComponent<Renderer>();
if (rend != null)
{
mat = rend.material;
}
}		 Start() Inisialisasi: mengambil komponen Renderer dari objek, lalu menyimpan material-nya ke variabel mat untuk akses cepat. Jika tidak ada renderer, mat tetap null dan metode lain akan aman (tidak error).
public void SetColor(Color color)
{
if (rend != null)
{
mat.color = color;
mat.EnableKeyword("_EMISSION");
mat.SetColor("_EmissionColor", color * 0.3f);
}
}		 SetColor() Fungsi: Mengubah warna stasiun dan menambahkan efek emisi (glow).
1. Jika renderer tersedia, ubah mat.color ke warna yang diberikan.
2. Aktifkan keyword _EMISSION pada shader material (agar emisi berfungsi).
3. Set _EmissionColor ke warna yang sama dengan intensitas 0.3x (glow redup).
Dipanggil oleh EvacuationManager.ChangeStationColor() saat stasiun selesai.
public void ResetColor()
{
if (rend != null)
{
mat.color = originalColor;
mat.DisableKeyword("_EMISSION");
}
}		 ResetColor() Fungsi: Mengembalikan warna stasiun ke warna asli dan menonaktifkan efek emisi.
1. Set mat.color ke originalColor (nilai yang diatur di Inspector).
2. Nonaktifkan keyword _EMISSION untuk menghilangkan glow.
Berguna jika ingin mereset stasiun ke keadaan awal (misalnya untuk restart scene).
// Dipasang pada setiap stasiun (IDCard, Alarm, Door, Exit) Penempatan Script ini ditempelkan pada keempat stasiun (IDCard, Alarm, Door, Exit) yang dibuat oleh EvacuationSceneBuilder. EvacuationManager memanggil SetColor(colorDone) saat stasiun selesai.
// Membuat material instance agar tidak mengganggu material asli Material Instance Karena rend.material menghasilkan instance material (bukan shared), perubahan hanya memengaruhi objek ini dan tidak mengganggu objek lain yang menggunakan material yang sama. Praktik terbaik untuk efek per objek.
