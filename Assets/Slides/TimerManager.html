⏱ TimerManager.cs

⏱ Timer Core · Runtime · 47 baris

Fungsi utama: Mengelola hitungan mundur (countdown timer) untuk evakuasi. Menampilkan waktu dalam format MM:SS, memicu event OnTimerExpiredEvent saat waktu habis, dan menyediakan metode untuk menghentikan atau mereset timer. Terintegrasi dengan EvacuationManager melalui event subscription untuk menangani kegagalan evakuasi.

📌 Baris Kode (cuplikan) 🎯 Fungsi / Metode 📖 Penjelasan
using UnityEngine;
using TMPro;
using System;		 Namespace Mengimpor namespace: UnityEngine untuk MonoBehaviour, TMPro untuk TextMeshPro, dan System untuk delegasi Action yang digunakan untuk event.
public class TimerManager : MonoBehaviour Deklarasi Kelas Kelas turunan MonoBehaviour yang dapat dipasang sebagai komponen pada GameObject di scene (biasanya objek "TimerManager" yang dibuat oleh scene builder).
public TextMeshProUGUI timerText;
public float maxTime = 120f;		 Variabel Publik timerText – referensi ke komponen TextMeshProUGUI untuk menampilkan waktu (biasanya "TimerText" di UI).
maxTime – durasi maksimum timer dalam detik (bisa diatur di Inspector). Default 120 detik (2 menit).
private float currentTime;
private bool isRunning = true;		 Variabel Privat currentTime – menyimpan sisa waktu saat ini (di-update setiap frame).
isRunning – status apakah timer sedang berjalan (true) atau berhenti (false).
public Action OnTimerExpiredEvent; Event Delegasi Action yang akan dipanggil saat timer mencapai 0. EvacuationManager mendaftarkan metode OnTimerExpired ke event ini di Start() untuk menangani kegagalan evakuasi.
private void Start()
{
currentTime = maxTime;
UpdateTimerDisplay();
}		 Start() Inisialisasi: set currentTime ke maxTime, lalu perbarui tampilan teks agar menunjukkan waktu awal (misal "02:00").
private void Update()
{
if (!isRunning) return;
currentTime -= Time.deltaTime;
if (currentTime <= 0f)
{
currentTime = 0f;
isRunning = false;
OnTimerExpiredEvent?.Invoke();
UpdateTimerDisplay();
}
else
{
UpdateTimerDisplay();
}
}		 Update() Logika timer:
1. Jika timer tidak berjalan (!isRunning), keluar dari metode.
2. Kurangi currentTime dengan Time.deltaTime setiap frame.
3. Jika waktu habis (currentTime <= 0):
  - Set currentTime = 0 (pastikan tidak negatif).
  - Set isRunning = false (hentikan timer).
  - Panggil OnTimerExpiredEvent?.Invoke() untuk memberi tahu subscriber.
  - Perbarui tampilan.
4. Jika waktu masih tersisa, perbarui tampilan setiap frame.
private void UpdateTimerDisplay()
{
if (timerText == null) return;
int minutes = Mathf.FloorToInt(currentTime / 60);
int seconds = Mathf.FloorToInt(currentTime % 60);
timerText.text = $"{minutes:00}:{seconds:00}";
}		 UpdateTimerDisplay() Memperbarui tampilan:
1. Jika timerText null, keluar (aman).
2. Hitung menit dan detik dari currentTime.
3. Format teks menjadi "MM:SS" dengan dua digit (misal "02:00", "01:30", "00:05").
Dipanggil setiap frame dari Update() dan saat inisialisasi.
public void StopTimer()
{
isRunning = false;
}		 StopTimer() Metode publik untuk menghentikan timer. Dipanggil oleh EvacuationManager saat evakuasi berhasil (InteractExitStation()) agar timer berhenti menghitung mundur.
public void ResetTimer()
{
currentTime = maxTime;
isRunning = true;
UpdateTimerDisplay();
}		 ResetTimer() Metode publik untuk mereset timer ke nilai awal (maxTime) dan menjalankannya kembali. Berguna untuk restart level atau permainan ulang.
// Dipisah dari EvacuationManager untuk Single Responsibility Desain TimerManager dipisah menjadi script terpisah (bukan bagian dari EvacuationManager) agar tanggung jawab lebih jelas (Single Responsibility Principle). EvacuationManager hanya menggunakan event OnTimerExpiredEvent untuk merespons.
