EvacuationSceneBuilder.cs Assets/Editor/
⚙ Editor Tool

Fungsi: Membangun seluruh scene secara otomatis dari kode. Membuat lighting, dinding, stasiun, UI overlay, player, manager, dan tombol world-space.

  • BuildScene() — Entry point menu Tools → Build Evacuation Scene
  • CreateWorldSpaceButton() — Buat tombol 3D dengan Canvas World Space
  • CreateStation() — Buat stasiun interaktif dengan label
  • CreateUIPanel() — Buat panel UI overlay
🔧 Editor · Build-time
EvacuationManager.cs Assets/Scripts/UTS_Dimas/
🧠 Core Logic

Fungsi: Otak utama permainan. Mengelola status evakuasi (ID card, alarm, pintu, exit), memperbarui UI checklist, minimap, feedback, dan menangani interaksi tombol.

  • InteractIDCardStation() — Ambil kartu ID
  • InteractAlarmStation() — Aktifkan alarm
  • InteractEmergencyDoorStation() — Buka pintu darurat
  • InteractExitStation() — Selesaikan evakuasi
  • UpdateChecklistUI() / UpdateMapUI() — Perbarui tampilan
  • ShowWarningPopup() / ShowSuccessFeedback() — Notifikasi
🎯 Runtime · Manager
TimerManager.cs Assets/Scripts/UTS_Dimas/
⏱ Timer

Fungsi: Mengelola hitungan mundur 120 detik. Memicu event OnTimerExpiredEvent saat waktu habis, dan memperbarui tampilan teks timer.

  • Update() — Kurangi waktu setiap frame
  • StopTimer() — Hentikan timer
  • ResetTimer() — Reset ke 120 detik
  • OnTimerExpiredEvent — Event untuk EvacuationManager
⏳ Runtime · Timer
DoorController.cs Assets/Scripts/UTS_Dimas/
🚪 Interaksi

Fungsi: Mengontrol pergerakan pintu. Saat dipanggil OpenDoor(), pintu bergeser ke posisi terbuka secara halus.

  • OpenDoor() — Buka pintu (geser)
  • CloseDoor() — Tutup pintu
  • IsOpen() — Cek status pintu
🚪 Runtime · Pintu
SimpleFirstPersonController.cs Assets/Scripts/UTS_Dimas/
🎮 Player

Fungsi: Mengontrol pergerakan pemain FPS. Menggunakan CharacterController untuk berjalan dan Mouse untuk melihat ke segala arah.

  • Update() — Baca input mouse & keyboard, gerakkan player
  • Start() — Inisialisasi, lock cursor
🎮 Runtime · Player
SpacebarClickHandler.cs Assets/Scripts/UTS_Dimas/
⌨ Input

Fungsi: Menangkap input Space dan melakukan raycast ke UI di tengah layar. Jika menemukan tombol, maka tombol tersebut diklik secara virtual.

  • Update() — Deteksi Space, raycast UI, invoke button
⌨ Runtime · Input
ButtonClickEffect.cs Assets/Scripts/UTS_Dimas/
✨ UI Effect

Fungsi: Memberikan efek visual pada tombol saat di-hover, ditekan, atau dilepas. Mengubah skala dan warna secara halus.

  • OnPointerEnter() — Hover: skala membesar, warna terang
  • OnPointerDown() — Press: skala mengecil, warna gelap
  • OnPointerUp() / OnPointerExit() — Kembali ke normal
✨ Runtime · UI
ButtonPulseEffect.cs Assets/Scripts/UTS_Dimas/
✨ UI Effect

Fungsi: Membuat tombol berdenyut (pulse) terus-menerus dengan skala naik-turun menggunakan Mathf.PingPong. Menarik perhatian pemain.

  • Pulse() — Coroutine yang mengubah skala secara periodik
✨ Runtime · UI
StationClickEffect.cs Assets/Scripts/UTS_Dimas/
🎨 Visual

Fungsi: Mengatur efek visual stasiun (warna dan emisi). Digunakan oleh EvacuationManager untuk mengubah warna stasiun saat selesai.

  • SetColor(Color) — Ubah warna material + emisi
  • ResetColor() — Kembalikan ke warna asli
🎨 Runtime · Visual
WarningPopup.cs Assets/Scripts/UTS_Dimas/
⚠ Popup

Fungsi: Menampilkan popup peringatan atau sukses dengan efek fade in/out. Juga menampilkan flash merah untuk peringatan error.

  • ShowWarning(string) — Tampilkan popup merah + flash
  • ShowSuccess(string) — Tampilkan popup hijau
  • AnimatePopup() — Coroutine fade in/out
⚠ Runtime · UI
EvacuationAutoConnector.cs Assets/Scripts/UTS_Dimas/
🔌 Auto-Connect

Fungsi: Berjalan di Edit Mode (ExecuteAlways) untuk secara otomatis menghubungkan referensi UI ke EvacuationManager. Memudahkan setup tanpa drag-drop manual.

  • ConnectComponents() — Cari semua Text, Image, CanvasGroup dan assign ke manager
