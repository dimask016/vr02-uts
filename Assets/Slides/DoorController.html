🚪 DoorController.cs

🚪 Interaksi · Runtime · 42 baris

Fungsi utama: Mengontrol pergerakan pintu darurat. Saat dipanggil OpenDoor(), pintu bergeser secara halus (menggunakan Vector3.Lerp) dari posisi tertutup ke posisi terbuka dengan offset yang dapat diatur. Mendukung metode CloseDoor() dan pengecekan status IsOpen().

📌 Baris Kode (cuplikan) 🎯 Fungsi / Metode 📖 Penjelasan
private Vector3 closedPosition;
private Vector3 openPosition;
public float openOffset = 2f;
private bool isOpen = false;
private float moveSpeed = 3f;		 Variabel closedPosition – posisi awal pintu (saat tertutup).
openPosition – posisi pintu saat terbuka (dihitung dari closedPosition + offset).
openOffset – jarak pergeseran pintu saat terbuka (bisa diatur di Inspector).
isOpen – status boolean apakah pintu sedang terbuka.
moveSpeed – kecepatan pergerakan pintu (semakin tinggi semakin cepat).
private void Start()
{
closedPosition = transform.position;
openPosition = closedPosition + new Vector3(0, 0, openOffset);
}		 Start() Inisialisasi: menyimpan posisi awal pintu sebagai closedPosition, lalu menghitung openPosition dengan menambahkan offset pada sumbu Z (arah depan pintu). Offset dapat diatur di Inspector.
private void Update()
{
if (isOpen)
{
transform.position = Vector3.Lerp(transform.position, openPosition, Time.deltaTime * moveSpeed);
}
}		 Update() Setiap frame, jika pintu dalam keadaan terbuka (isOpen == true), posisi pintu di-lerp menuju openPosition dengan kecepatan moveSpeed. Memberikan efek pergerakan yang halus.
public void OpenDoor()
{
if (!isOpen)
{
isOpen = true;
Debug.Log("[DoorController] Door opened!");
}
}		 OpenDoor() Metode publik untuk membuka pintu. Memeriksa apakah pintu belum terbuka, jika ya maka set isOpen = true sehingga Update() mulai menggerakkan pintu. Mencetak log untuk debugging.
public void CloseDoor()
{
if (isOpen)
{
isOpen = false;
Debug.Log("[DoorController] Door closed!");
}
}		 CloseDoor() Metode publik untuk menutup pintu. Memeriksa apakah pintu sedang terbuka, jika ya maka set isOpen = false sehingga pergerakan berhenti. Pintu akan tetap di posisi terakhirnya (tidak otomatis kembali).
public bool IsOpen()
{
return isOpen;
}		 IsOpen() Metode publik untuk mengecek status pintu. Mengembalikan nilai isOpen. Berguna bagi script lain (misalnya EvacuationManager) untuk mengetahui apakah pintu sudah terbuka.
// Pintu tidak otomatis kembali ke posisi tertutup Desain Pintu hanya bergerak saat isOpen == true. Saat CloseDoor() dipanggil, pintu berhenti di posisi terakhirnya. Untuk mengembalikan ke posisi tertutup, perlu implementasi tambahan (misalnya lerp balik ke closedPosition).
// Dipanggil dari EvacuationManager.InteractEmergencyDoorStation() Integrasi Script ini digunakan oleh EvacuationManager saat pemain mengklik stasiun pintu (InteractEmergencyDoorStation()). Setelah semua syarat terpenuhi, door.OpenDoor() dipanggil untuk membuka pintu.
