🎮 SimpleFirstPersonController.cs

🎮 Player Controller · Runtime · 49 baris

Fungsi utama: Mengontrol pergerakan pemain dalam mode First-Person Shooter (FPS). Menggunakan CharacterController untuk bergerak dan input Mouse untuk rotasi kamera. Mendukung berjalan (WASD), melihat (mouse), serta penguncian kursor untuk pengalaman FPS yang imersif. Dirancang sederhana dan ringan, cocok untuk proyek skala kecil seperti simulasi evakuasi.

public Transform playerCamera;
public CharacterController characterController;
public float walkSpeed = 5f;
public float mouseSensitivity = 2f;
private float verticalRotation = 0f;		 Variabel playerCamera – referensi ke transform kamera (biasanya anak dari GameObject player).
characterController – komponen CharacterController untuk pergerakan fisik.
walkSpeed – kecepatan berjalan (dapat diatur di Inspector).
mouseSensitivity – sensitivitas mouse (semakin tinggi semakin cepat rotasi).
verticalRotation – menyimpan rotasi vertikal kamera (pitch) untuk di-clamp agar tidak over-rotate.
private void Start()
{
if (characterController == null)
characterController = GetComponent<CharacterController>();
if (playerCamera == null && GetComponentInChildren<Camera>() != null)
playerCamera = GetComponentInChildren<Camera>().transform;
Cursor.lockState = CursorLockMode.Locked;
Cursor.visible = false;
}		 Start() Inisialisasi:
1. Jika characterController belum di-set, cari komponen CharacterController pada GameObject yang sama.
2. Jika playerCamera belum di-set, cari komponen Camera di anak-anak objek dan ambil transform-nya.
3. Kunci kursor ke tengah layar (CursorLockMode.Locked) dan sembunyikan kursor untuk pengalaman FPS.
float mouseX = Input.GetAxis("Mouse X") * mouseSensitivity;
float mouseY = Input.GetAxis("Mouse Y") * mouseSensitivity;

verticalRotation -= mouseY;
verticalRotation = Mathf.Clamp(verticalRotation, -90f, 90f);
playerCamera.localRotation = Quaternion.Euler(verticalRotation, 0f, 0f);
transform.Rotate(Vector3.up * mouseX);		 Mouse Look Rotasi kamera:
1. Baca input mouse horizontal (Mouse X) dan vertikal (Mouse Y), kalikan dengan sensitivitas.
2. Rotasi vertikal: kurangi verticalRotation dengan mouseY (karena gerakan mouse ke atas harus membuat kamera melihat ke bawah).
3. Mathf.Clamp membatasi sudut pitch antara -90° dan +90° agar pemain tidak bisa melihat ke belakang (ke arah bawah/atas ekstrem).
4. Terapkan rotasi vertikal ke playerCamera.localRotation (hanya sumbu X).
5. Rotasi horizontal: putar transform (body player) di sumbu Y sebesar mouseX.
float horizontal = Input.GetAxis("Horizontal");
float vertical = Input.GetAxis("Vertical");
Vector3 move = transform.right * horizontal + transform.forward * vertical;
move.Normalize();
characterController.Move(move * walkSpeed * Time.deltaTime);		 Movement Pergerakan:
1. Baca input horizontal (A/D atau kiri/kanan) dan vertikal (W/S atau atas/bawah).
2. Hitung arah gerakan relatif terhadap rotasi player (transform.right dan transform.forward) agar gerakan selalu sesuai dengan arah hadap pemain.
3. Normalize() memastikan kecepatan diagonal tidak lebih cepat dari gerakan lurus (mencegah speed boost).
4. characterController.Move menggerakkan pemain dengan kecepatan walkSpeed dikalikan Time.deltaTime.
private void OnApplicationFocus(bool hasFocus)
{
if (hasFocus)
{
Cursor.lockState = CursorLockMode.Locked;
Cursor.visible = false;
}
}		 OnApplicationFocus() Dipanggil saat aplikasi mendapatkan atau kehilangan fokus (misalnya saat alt-tab). Jika aplikasi mendapatkan fokus kembali, kunci dan sembunyikan kursor agar pemain tidak kehilangan kontrol FPS.
