🔌 EvacuationAutoConnector.cs

🔧 Auto-Connect · Edit-time · 54 baris

Fungsi utama: Menghubungkan secara otomatis semua komponen UI (TextMeshPro, Image, CanvasGroup) ke EvacuationManager berdasarkan nama GameObject. Berjalan hanya di Edit Mode (bukan saat Play) berkat atribut [ExecuteAlways] dan preprocessor #if UNITY_EDITOR. Memudahkan proses setup scene tanpa perlu drag-drop manual.

📌 Baris Kode (cuplikan) 🎯 Fungsi / Metode 📖 Penjelasan
[ExecuteAlways] Atribut Menandai bahwa script ini akan berjalan baik di Edit Mode maupun Play Mode. Memungkinkan metode Start() dipanggil di editor saat objek diaktifkan atau scene dimuat.
public class EvacuationAutoConnector : MonoBehaviour Deklarasi Kelas Kelas turunan MonoBehaviour yang dapat dipasang sebagai komponen pada GameObject mana pun di scene (biasanya pada EvacuationManager atau objek kosong).
void Start()
{
#if UNITY_EDITOR
if (!Application.isPlaying)
{
ConnectComponents();
}
#endif
}		 Start() Logika:
1. Preprocessor #if UNITY_EDITOR memastikan kode hanya dieksekusi di dalam Editor Unity (tidak di build).
2. !Application.isPlaying memastikan script hanya berjalan saat Edit Mode, bukan saat Play.
3. Jika kondisi terpenuhi, panggil ConnectComponents() untuk menghubungkan semua komponen.
void ConnectComponents()
{
EvacuationManager evac = FindObjectOfType<EvacuationManager>();
if (evac == null) return;		 ConnectComponents() Mencari EvacuationManager di scene. Jika tidak ditemukan, hentikan eksekusi (return) karena tidak ada yang perlu dihubungkan.
TextMeshProUGUI[] allTexts = FindObjectsOfType<TextMeshProUGUI>();
Image[] allImages = FindObjectsOfType<Image>();
CanvasGroup[] allCanvasGroups = FindObjectsOfType<CanvasGroup>();		 Find Components Mengambil semua komponen TextMeshProUGUI, Image, dan CanvasGroup yang ada di seluruh scene (termasuk yang tidak aktif).
foreach (var text in allTexts)
{
if (text.gameObject.name == "Txt_ID") evac.txtObjectiveID = text;
if (text.gameObject.name == "Txt_Alarm") evac.txtObjectiveAlarm = text;
if (text.gameObject.name == "Txt_Door") evac.txtObjectiveDoor = text;
if (text.gameObject.name == "Txt_Exit") evac.txtObjectiveExit = text;
if (text.gameObject.name == "TimerText") evac.txtTimer = text;
if (text.gameObject.name == "StatusText") evac.txtFeedback = text;
if (text.gameObject.name == "PopupText") evac.warningPopupText = text;
}		 Connect Text Iterasi semua TextMeshProUGUI dan mencocokkan gameObject.name dengan nama yang diharapkan. Jika cocok, assign ke field yang sesuai di EvacuationManager.
foreach (var image in allImages)
{
if (image.gameObject.name == "Light_ID") evac.mapIDCard = image;
if (image.gameObject.name == "Light_Alarm") evac.mapAlarm = image;
if (image.gameObject.name == "Light_Door") evac.mapDoor = image;
if (image.gameObject.name == "Light_Exit") evac.mapExit = image;
if (image.gameObject.name == "WarningFlash") evac.flashImage = image;
}		 Connect Image Iterasi semua Image dan mencocokkan nama GameObject dengan ikon minimap (Light_ID, Light_Alarm, dll.) serta WarningFlash untuk efek kilatan.
foreach (var cg in allCanvasGroups)
{
if (cg.gameObject.name == "WarningPopupPanel")
evac.warningCanvasGroup = cg;
}		 Connect CanvasGroup Iterasi semua CanvasGroup dan mencari yang bernama WarningPopupPanel untuk diassign ke evac.warningCanvasGroup (digunakan untuk mengontrol alpha popup).
Debug.Log("EvacuationAutoConnector: Components connected successfully!"); Log Mencetak pesan ke Console Unity sebagai konfirmasi bahwa proses koneksi berhasil dijalankan.
// (seluruh kode) Catatan Penting 1. Script ini tidak berjalan di build/standalone karena preprocessor #if UNITY_EDITOR.
2. Sangat berguna untuk scene yang dibangun secara prosedural (seperti oleh EvacuationSceneBuilder) agar referensi UI otomatis terhubung tanpa intervensi manual.
3. Jika ada perubahan nama GameObject, script ini harus diperbarui agar tetap berfungsi.
