⚠ WarningPopup.cs

⚠ Popup UI · Runtime · 72 baris

Fungsi utama: Menampilkan popup peringatan atau notifikasi sukses dengan efek animasi fade in/out. Mendukung flash merah untuk peringatan error dan manajemen coroutine untuk mencegah tumpang tindih animasi. Digunakan oleh EvacuationManager untuk memberi umpan balik visual yang jelas kepada pemain.

using UnityEngine;
using UnityEngine.UI;
using TMPro;
using System.Collections;		 Namespace Mengimpor namespace: UnityEngine untuk MonoBehaviour, UnityEngine.UI untuk Image, TMPro untuk TextMeshPro, dan System.Collections untuk Coroutine (IEnumerator).
public class WarningPopup : MonoBehaviour Deklarasi Kelas Kelas turunan MonoBehaviour yang dapat dipasang pada GameObject popup (biasanya "WarningPopupPanel" di UI).
public CanvasGroup canvasGroup;
public TMP_Text messageText;
public Image flashImage;
public float fadeDuration = 0.5f;
public float displayDuration = 2f;
private Coroutine activeRoutine;		 Variabel canvasGroup – komponen CanvasGroup untuk mengontrol alpha (transparansi) seluruh popup.
messageText – komponen TMP_Text untuk menampilkan pesan.
flashImage – komponen Image untuk efek kilatan merah (full-screen overlay).
fadeDuration – durasi animasi fade in/out (bisa diatur di Inspector).
displayDuration – durasi popup ditampilkan sebelum fade out.
activeRoutine – menyimpan referensi coroutine yang sedang berjalan untuk menghentikannya jika perlu.
public void ShowWarning(string message)
{
if (messageText != null) messageText.text = message;
if (activeRoutine != null) StopCoroutine(activeRoutine);
activeRoutine = StartCoroutine(AnimatePopup());
StartCoroutine(FlashRed());
}		 ShowWarning() Fungsi: Menampilkan popup peringatan (error).
1. Set teks pesan.
2. Hentikan coroutine sebelumnya jika ada (mencegah konflik animasi).
3. Mulai coroutine AnimatePopup() untuk fade in/out.
4. Mulai coroutine FlashRed() untuk efek kilatan merah.
Dipanggil oleh EvacuationManager saat terjadi error (misal waktu habis, syarat belum terpenuhi).
public void ShowSuccess(string message)
{
if (messageText != null) messageText.text = message;
if (activeRoutine != null) StopCoroutine(activeRoutine);
activeRoutine = StartCoroutine(AnimatePopup());
}		 ShowSuccess() Fungsi: Menampilkan popup sukses (tanpa flash merah).
1. Set teks pesan.
2. Hentikan coroutine sebelumnya jika ada.
3. Mulai coroutine AnimatePopup() untuk fade in/out.
Berbeda dengan ShowWarning(), tidak memicu FlashRed().
private IEnumerator AnimatePopup()
{
float elapsed = 0f;
while (elapsed < fadeDuration)
{
canvasGroup.alpha = Mathf.Lerp(0f, 1f, elapsed / fadeDuration);
elapsed += Time.deltaTime;
yield return null;
}
canvasGroup.alpha = 1f;
yield return new WaitForSeconds(displayDuration);
elapsed = 0f;
while (elapsed < fadeDuration)
{
canvasGroup.alpha = Mathf.Lerp(1f, 0f, elapsed / fadeDuration);
elapsed += Time.deltaTime;
yield return null;
}
canvasGroup.alpha = 0f;
}		 AnimatePopup() – Coroutine Logika animasi fade:
1. Fade in: alpha dari 0 ke 1 selama fadeDuration detik.
2. Tunggu selama displayDuration detik (popup terlihat penuh).
3. Fade out: alpha dari 1 ke 0 selama fadeDuration detik.
4. Set alpha = 0 (pastikan popup benar-benar tersembunyi).
Menggunakan yield return null untuk menunggu setiap frame.
private IEnumerator FlashRed()
{
if (flashImage == null) yield break;
float duration = 0.2f, elapsed = 0f;
while (elapsed < duration)
{
flashImage.color = new Color(1, 0, 0, Mathf.Lerp(0f, 0.6f, elapsed / duration));
elapsed += Time.deltaTime;
yield return null;
}
elapsed = 0f;
while (elapsed < duration)
{
flashImage.color = new Color(1, 0, 0, Mathf.Lerp(0.6f, 0f, elapsed / duration));
elapsed += Time.deltaTime;
yield return null;
}
flashImage.color = new Color(1, 0, 0, 0);
}		 FlashRed() – Coroutine Logika kilatan merah:
1. Jika flashImage null, hentikan coroutine (yield break).
2. Fade in merah: alpha dari 0 ke 0.6 selama 0.2 detik.
3. Fade out merah: alpha dari 0.6 ke 0 selama 0.2 detik.
4. Set alpha = 0 (pastikan flash hilang).
Efek ini menciptakan kilatan cepat di layar untuk menarik perhatian pemain saat terjadi error.
// Dipanggil dari EvacuationManager.ShowWarningPopup() Integrasi Script ini biasanya dipasang pada objek "WarningPopupPanel" yang dibuat oleh EvacuationSceneBuilder. EvacuationManager memanggil ShowWarning() atau ShowSuccess() melalui referensi warningPopupText dan warningCanvasGroup.
// activeRoutine mencegah tumpang tindih animasi Manajemen Variabel activeRoutine menyimpan coroutine yang sedang berjalan. Saat ShowWarning() atau ShowSuccess() dipanggil, coroutine lama dihentikan (StopCoroutine) sebelum yang baru dimulai. Mencegah tumpang tindih animasi jika popup dipanggil berulang kali.
