⌨ SpacebarClickHandler.cs

⌨ Input Handler · Runtime · 28 baris

Fungsi utama: Menangkap input tombol Space dan mensimulasikan klik pada tombol UI yang berada tepat di tengah layar (crosshair). Menggunakan sistem EventSystem dan Raycast untuk mendeteksi komponen Button di bawah kursor (tengah layar), lalu memicu onClick-nya. Memungkinkan pemain berinteraksi dengan tombol world-space tanpa harus mengarahkan kursor secara manual.

📌 Baris Kode (cuplikan) 🎯 Fungsi / Metode 📖 Penjelasan
using UnityEngine;
using UnityEngine.EventSystems;
using UnityEngine.UI;		 Namespace Mengimpor namespace yang diperlukan: UnityEngine untuk MonoBehaviour, UnityEngine.EventSystems untuk sistem raycast UI, dan UnityEngine.UI untuk komponen Button.
public class SpacebarClickHandler : MonoBehaviour Deklarasi Kelas Kelas turunan MonoBehaviour yang dapat dipasang pada objek kamera atau objek lain di scene. Biasanya ditempelkan pada MainCamera agar selalu aktif saat pemain bergerak.
void Update()
{
if (Input.GetKeyDown(KeyCode.Space))
{
// ...
}
}		 Update() Setiap frame, periksa apakah tombol Space baru saja ditekan (GetKeyDown). Jika ya, jalankan logika raycast untuk menemukan tombol di tengah layar.
PointerEventData pointerData = new PointerEventData(EventSystem.current);
pointerData.position = new Vector2(Screen.width / 2, Screen.height / 2);		 Pointer Data Buat objek PointerEventData yang merepresentasikan posisi pointer (kursor). Posisi diatur ke koordinat tengah layar (Screen.width/2, Screen.height/2) sehingga raycast akan mengenai elemen UI yang berada di titik tengah (di bawah crosshair).
var results = new System.Collections.Generic.List<RaycastResult>();
EventSystem.current.RaycastAll(pointerData, results);		 Raycast UI Buat daftar kosong untuk menampung hasil raycast, lalu panggil EventSystem.current.RaycastAll dengan pointerData sebagai parameter. Fungsi ini akan mengembalikan semua elemen UI (Graphic) yang berada di bawah posisi pointer (tengah layar).
foreach (var result in results)
{
Button btn = result.gameObject.GetComponent<Button>();
if (btn != null && btn.interactable)
{
btn.onClick.Invoke();
Debug.Log("Spacebar clicked: " + btn.name);
break;
}
}		 Deteksi & Klik Proses:
1. Iterasi semua RaycastResult yang ditemukan.
2. Ambil komponen Button dari GameObject hasil raycast.
3. Jika tombol ditemukan dan dalam keadaan interactable (aktif), panggil btn.onClick.Invoke() untuk menjalankan semua listener yang terdaftar.
4. Cetak log untuk debugging.
5. break setelah menemukan tombol pertama yang valid (prioritas tombol teratas).
// Dipasang pada MainCamera Penempatan Script ini sebaiknya ditempelkan pada MainCamera (atau objek lain yang selalu aktif) agar selalu mendeteksi input Space selama permainan berjalan. Dalam proyek ini, dipasang pada kamera yang menjadi anak dari Player.
// Bekerja dengan Canvas World Space World-space UI Script ini bekerja dengan semua jenis Canvas (Screen Space dan World Space) selama elemen UI memiliki komponen GraphicRaycaster dan Button. Sangat berguna untuk mengklik tombol 3D yang mengambang di dunia tanpa harus menggerakkan kursor.
