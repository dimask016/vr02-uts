# vr02-uts
Emergency Evacuation Training Panel  Buat simulasi evakuasi sederhana.
Soal 10: Emergency Evacuation Training Panel
Level: Sulit
Maksimal: 120
Buat simulasi evakuasi sederhana. Player berada di gedung latihan dan harus mengikuti checklist keselamatan melalui panel World Space Canvas.
Aturan minimal:
Scene memiliki minimal 4 station: ID Card Station, Alarm Panel, Emergency Door Panel, dan Exit Panel.
Setiap station memiliki Button World Space untuk menjalankan aksi.
Panel utama menampilkan checklist minimal 4 objektif: Ambil ID Card, Aktifkan Alarm, Buka Emergency Door, Pergi ke Exit.
Saat tombol pada station ditekan, checklist ter-update.
Emergency Door hanya bisa dibuka setelah ID Card diambil dan alarm diaktifkan.
Exit hanya bisa diselesaikan setelah Emergency Door terbuka.
UI memberi feedback jika urutan aksi salah, misalnya “Take ID Card First”, “Activate Alarm First”, atau “Emergency Door Opened”.
Ada warning visual sederhana menggunakan Image Color/Alpha atau status text.
Dinilai khusus — 40 poin
Scene memiliki minimal 4 station: ID Card, Alarm, Emergency Door, dan Exit — 5 poin
Setiap station memiliki Button World Space untuk menjalankan aksi — 6 poin
Checklist objektif ter-update sesuai aksi yang dilakukan player — 8 poin
Urutan aksi evakuasi berjalan sesuai requirement — 8 poin
Emergency Door hanya bisa dibuka setelah ID Card diambil dan alarm diaktifkan — 5 poin
Exit hanya bisa diselesaikan setelah Emergency Door terbuka — 4 poin
Feedback gagal/berhasil seperti “Take ID Card First” atau “Evacuation Complete” tampil jelas — 3 poin
Warning visual sederhana menggunakan Image Color/Alpha atau status text berhasil diterapkan — 1 poin
